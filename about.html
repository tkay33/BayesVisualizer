About

Hello!

This application provides recommendations for the best diagnostic test based on the naive Bayes Algorithm using prevalence, specificity and sensitivity of the disease. What it means is that you no longer have to make complex computations in your head to suggest diagnostic tests to patients. You will get a visualization which highlights the cost and risks involved for the specific diagnostic tests related to the disease you selected.

If you have any problems using the application or you encounter any errors, please submit your query in the contact page.
